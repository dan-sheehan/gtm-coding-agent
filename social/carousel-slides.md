## Slide 3 — The Shift

**Header:** coding agents are not chatbots

**Body:**

a chatbot answers questions.

a coding agent:
- reads your files
- runs commands
- writes scripts
- builds things
- remembers context across sessions

same AI. completely different workflow.

## Slide 4 — The System

**Body:**

```
gtm-os/

  messaging/    ← attack angles, value props
  segments/     ← target account lists
```

the agent reads these before every task.
